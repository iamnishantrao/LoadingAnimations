# After downloading?
1. Go to Project directory in Terminal and run "pod install". Make sure that you have CocoaPods installed on your system.
2. Open “.xcworkspace” in Xcode.
3. Build and Run the project.

# Requirements
1. Swift 3.0
2. Xcode 8.0+
3. Cocoapods 1.2.1
